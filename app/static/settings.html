Quick Token Setup

Run this one-liner in your Proxmox host shell (not inside the LXC). It creates a read-only monitoring user and API token, then prints the credentials to paste below.

bash <(curl -fsSL https://raw.githubusercontent.com/Mati-l33t/lan-tracker-network-sonar/main/proxmox/setup-token.sh)

Creates monitoring@pam user with LanTrackerRole — read-only + VM power management. Token ID will be monitoring@pam!lan-tracker. Safe to re-run.
